# algorithms_and_logic

A collection of classic algorithm and logic problems solved in pure Python — no libraries.

---

## Tools

| File | What it does |
|------|-------------|
| `armstrong_number_checker.py` | Checks if a number is an Armstrong number |
| `caesar_cipher.py` | Encrypts text using a Caesar cipher shift |
| `collatz_conjecture.py` | Generates the Collatz sequence for a number |
| `narcissistic_number_finder.py` | Finds all narcissistic numbers up to 10,000 |
| `number_guessing_game.py` | Binary search based number guessing game |
| `palindrome.py` | Checks if a word is a palindrome |
| `perfect_number_finder.py` | Finds all perfect numbers up to 1000 |
| `prime_numbers.py` | Early attempt at a prime number checker |
| `prime_number_2.py` | Refined version of the prime number checker |
| `Tribonacci.py` | Generates a Tribonacci sequence |
| `main.py` | Finds pairs of numbers from a list that sum to a target value |

## Philosophy

No libraries. Every algorithm written from first principles — including earlier, imperfect attempts kept to show iteration.
